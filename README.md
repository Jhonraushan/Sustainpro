# SustainPro Energy & Environment Services Pvt Ltd

Leading the Way to Eco-Excellence — Your partner in sustainable transformation.

## About

SustainPro is a pioneer specializing in GHG accounting, carbon neutrality, net zero roadmap design, and alignment with Science-Based Targets (SBTi). Based in Indore, Madhya Pradesh, India.

## Services

- **GHG Accounting** — Measurement, reporting, and management of greenhouse gas emissions
- **Carbon Neutrality** — Tailored strategies to offset emissions and achieve carbon neutrality
- **Net Zero Roadmap** — Science-based net zero strategies aligned with SBTi
- **SBTi Alignment** — Science-based targets driving meaningful emission reductions
- **Standards & Reporting** — ISO 14064, CDP, GRI, BRSR, SASB, and TCFD expertise
- **Decarbonization Roadmap** — Low-carbon transition planning with innovative technologies

## Tech Stack

- **React 19** — Component-based UI
- **Vite** — Fast build tool and dev server
- **CSS3** — Custom properties, grid layouts, animations
- **Font Awesome** — Service and team icons
- **Ionicons** — Social media icons

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens at `http://localhost:5173`

### Build

```bash
npm run build
```

Output goes to `dist/` folder.

### Preview Production Build

```bash
npm run preview
```
